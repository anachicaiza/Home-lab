# Help Desk Troubleshooting Scenarios

## Scenario 1 — User Cannot Log In

### User Issue

"I cannot log into my account."

### Troubleshooting Steps

1. Verify username
2. Check CAPS LOCK
3. Confirm password attempts
4. Verify account status
5. Check whether account is locked
6. Escalate if necessary

### Possible Causes

* Incorrect password
* Locked account
* Expired password


## Scenario 2 — Outlook Not Opening

### User Issue

"Outlook will not open."

### Troubleshooting Steps

1. Restart Outlook
2. Restart computer
3. Verify internet connection
4. Open Outlook in safe mode
5. Check for updates
6. Reinstall if required

### Possible Causes

* Corrupted profile
* Add-in issue
* Software problem


## Scenario 3 — VPN Not Connecting

### User Issue

"My VPN is not connecting."

### Troubleshooting Steps

1. Verify internet access
2. Check VPN credentials
3. Restart VPN client
4. Verify VPN server status
5. Escalate if necessary

### Possible Causes

* Incorrect credentials
* Internet connectivity issue
* VPN server outage


# Lessons Learned

Structured troubleshooting improves efficiency and helps identify issues logically rather than guessing randomly.
Communication and documentation are important throughout the troubleshooting process.
