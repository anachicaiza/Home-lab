# Windows Troubleshooting Scenarios

## Scenario 1 — Slow Computer

### User Issue

"My computer is running very slowly."

### Troubleshooting Steps

1. Open Task Manager
2. Check CPU and RAM usage
3. Identify unnecessary processes
4. Check startup applications
5. Restart system if required

### Possible Causes

* High CPU usage
* Too many startup applications
* Low memory availability



## Scenario 2 — No Internet Connection

### User Issue

"I cannot access the internet."

### Troubleshooting Steps

1. Verify physical connection
2. Run ipconfig
3. Ping gateway
4. Test internet connectivity
5. Check DNS using nslookup
6. Restart network adapter if necessary

### Possible Causes

* Invalid IP configuration
* DNS issue
* Disabled network adapter



## Scenario 3 — Printer Not Working

### User Issue

"My printer is not printing."

### Troubleshooting Steps

1. Verify printer is powered on
2. Check cable/network connection
3. Restart Print Spooler service
4. Remove and re-add printer
5. Perform test print

### Possible Causes

* Print queue issue
* Printer offline
* Driver problem
